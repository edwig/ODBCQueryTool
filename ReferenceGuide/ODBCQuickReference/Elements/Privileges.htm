Privileges

SYSTEM

System Privilege

Allows Grantee To ...

USAGE DOMAIN

Allows a grantee to create columns with this domain

USAGE CHARACTER SET

Allows a grantee to create columns with this character set

USAGE COLLATION

Allows a grantee to create columns with this collation and/or to sort a selection in this collation

USAGE TRANSLATION

Allows a grantee to use a translation from one character set to another

TABLES AND/OR VIEWS

System Privilege

Allows Grantee To ...

INSERT TABLE

Allows grantee to insert rows in a table or view

SELECT TABLE

Allows grantee to select rows from a table or view

UPDATE TABLE

Allows grantee to update rows in a table or view

DELETE TABLE

Allows grantee to delete rows from a table or view

REFERENCES TABLE

Allows grantee to create foreign keys in other tables that references rows in this table

COLUMNS

System Privilege

Allows Grantee To ...

INSERT COLUMN

Allows grantee to insert new values in this column in a table row constructor

UPDATE COLUMN

Allows grantee to update columns of a row in this table

REFERENCES COLUMN

Allows grantee to create foreign keys in other tables that reference to this column in this table
 
To see which type of system privileges are supported, look them up in the ODBC tree under 'SQL Supported standard' under either 'Grant' or 'Revoke'.
