column-name

A column-name is a reference to a column in the current table or of one of the specified tables in the FROM clause of a SELECT statement.

Support

Predicate type

Minimum

[table-name.]column-identifier

Core

[{table-name | correlation-name}.]column-identifier
 
correlation-name ::= user defined name in the 'AS' option of the FROM clause
table-name and column-identifier are user defined identifiers
