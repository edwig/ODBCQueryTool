pattern

A pattern value is used in a LIKE search-condition. A pattern consists either of:

1: A character-string-literal: '{character}'. To include a single literal quote, use two quotes;

2: A dynamic-parameter: the '?' character for use in binded parameters.

3: The code-word 'USER'

In the case of a LIKE pattern in the character-string-literal the percent character (%)   matches zero or more of any characters; the underscore character (_) matches 1 character.
