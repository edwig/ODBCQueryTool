CREATE SCHEMA

Use this command to create multiple tables and views and perform multiple grants in a single transaction.

 

Syntax of the SQL statement

CREATE SCHEMA schema-name-clause

[DEFAULT CHARACTER SET character-set]

{

   create-domain

| create-index

| create-table

| create-view

| grant-statement

| [,...]

}

;

schema-name-clause ::=

 

   schema-name

| schema-name | AUTHORIZATION authorization-identifier

| AUTHORIZATION authorization-identifier
 

To see if a particular syntax is supported consult the ODBC tree for the following items:

- The SQL conformance level

- The ODBC conformance level

- 'Create   schema' under 'SQL support'. This info is rather unreliable for most database vendors.
