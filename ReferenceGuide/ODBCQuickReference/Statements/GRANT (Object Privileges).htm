GRANT (Object Privileges)

Use this command to grant privileges for a particular table or column to users. To grant system privileges, use the GRANT command (System Privileges).

Syntax of the SQL statement

GRANT { object-privilege [,...] | ALL [PRIVILEGES]}

ON base-table-name TO user-name [,...]

[ WITH GRANT OPTION]

;

object-privilege ::=

 

    table-privilege

| column-privilege

table-privilege ::=

 

   INSERT

| SELECT

| UPDATE

| DELETE

| REFERENCES [( column [, ... ])]

column-privilege ::=

 

   INSERT

| UPDATE

| REFERENCES [( column [, ... ])]

user-name ::=

 

letter { letter | digit | '_' [, ...] }

 

Too see which options of the grant statement are supported, look up the grant statement in the ODBC Info tree under 'SQL Supported standard / Grant'.
