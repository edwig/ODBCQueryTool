GRANT (System Privileges)

Use this command to grant system privileges to users. To grant object privileges, use the GRANT command (Object Privileges).

Syntax of the SQL statement

GRANT USAGE

ON system-object TO user-name [,...]

[ WITH GRANT OPTION]

;

system-object ::=

 

   CHARACTER SET

| COLLATION

| DOMAIN

| TRANSLATION

user-name ::=

 

letter { letter | digit | '_' [, ...] }

 

Too see which options of the grant statement are supported, look up the grant statement in the ODBC Info tree under 'SQL Supported standard / Grant'.
