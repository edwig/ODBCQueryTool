UPDATE

Use this command to change existing values in a table or in a view's base table.

Syntax of the SQL statement

UPDATE base-table-name

    SET column = expression [,...]

[WHERE search-condition]

;

UPDATE base-table-name

    SET (column [,...]) = {( expression [,...]) | sub-query }

[WHERE search-condition]

;

 

The second form is supported on **some** databases and can be quicker to update multiple rows from one query instead of more queries.
