Querying

Querying from within the editor is a breeze. Write your SQL commands and terminate them with a ';' or a '/' sign.

Then from within the first line of the SQL commmand, press F10 to execute the current command, or F11 to execute the command and simultanously move on the the next command.

All output shoul be shown in the lower panes. There are four different panes:

1) Query: All the query output lines are shown here;

2) Output: Result of the queries. OK's or errors are shown here;

3) Statistics: Timing of the executed queries;

4) History: All the last executed queries. Handy for cut-and-paste purposes.

The query result pane also contains buttons for paging through the results or for exporting the query results to a HTML browser view or an Excel spreadsheet view.
