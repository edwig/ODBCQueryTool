The session and script menu's are briefly documented here, because they perform functions for quering of databases.

The Session menu

The session menu contains the following functions

Menu item

Function

Connect

Connect to a different database
Disconnect

Disconnect from the current database
Begin

Begin a new transaction. Normally the ODBC connection is in auto-commit mode, so every query will be committed on it's own. Use the 'begin' function to delay the commit till the end of all queries.
Commit

Commit your changes. All changes from the point were you used the 'Begin' command will be permanently commited to your database.
Rollback

Rollback and forget all changes since the point were you used the 'Begin' command. No changes will be made to the database.
ODBC Report

Writes a report of all ODBC properties. All the properties from the ODBC properties panel are shown in this report. The report serves as a basis to compare one or more ODBC datasources in capabilities.
Session status

Shows the current session status. This status is the same as the details window from the connection dialog at startup.
