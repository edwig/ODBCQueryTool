Welcome to the ODBC Querytool

This guide describes the functions of the EDO 1) Open ODBC Querytool in three parts:

  • The editor: Writing SQL in text files of type *.SQL
  • Querying from within the editor: Using the editor to run your queries.
  • ODBC SQL Quick reference: The greatest common denominator of SQL grammar
